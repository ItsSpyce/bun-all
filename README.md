# `bun-all`

A supercharged way to run scripts in bun!

## Usage

```bash
# add the following scripts
# "scripts": {
#   "build": {
#     "dev": "echo 'Hello from dev :)'"
#     "prod": "echo 'Hello from prod D:'"
#   }
# }
#
#
bunx bun-all build
# => 'Hello from dev :)'
# => 'Hello from prod D:'
```
